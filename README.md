# Console Todo App (Phase I)

A robust, interactive CLI Todo application built with Python, Rich, and Pydantic.

## Setup

Ensure you have Python 3.13+ installed.

1.  Install dependencies (using `uv` or `pip`):
    ```bash
    pip install -e .[dev]
    ```

## Usage

Run the application:
```bash
python src/main.py
```

## Testing

Run tests with coverage:
```bash
pytest
```
